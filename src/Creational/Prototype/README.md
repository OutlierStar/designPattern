然后，我们再次调用 clone() 方法克隆出另一个副本 clone2，并输出其属性值。我们可以看到 clone2 的属性值与原型对象的属性值相同，验证了克隆过程成功。

通过原型模式，我们可以通过克隆来创建新的对象，而无需依赖具体类的构造函数。这种方式更加灵活和高效，特别在对象的创建成本较高或需要动态创建对象时非常有用。
